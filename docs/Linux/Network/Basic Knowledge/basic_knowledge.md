## 协议
协议就是计算机与计算机之间通过网络实现通信时事先达成的一种“约定”。这种“约定”使那些由不同厂商的设备、不同的CPU以及不同的操作系统组成的计算机之间，只要遵循相同的协议就能实现通信。

协议可以分为很多种，每一种协议都明确界定了它的行为规范。

两台计算机之间必须能够支持相同的协议，并遵循相同协议进行处理，才能实现相互通信。
